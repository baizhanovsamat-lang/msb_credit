# 📱 Руководство: Развёртывание МСБ Кредит как PWA

## Что получите в итоге
- Ссылка типа `msb-credit.vercel.app`
- Менеджеры устанавливают на iPhone/Android как приложение (без App Store)
- Работает офлайн (просматривать данные без интернета)
- Бесплатно

---

## ШАГ 1 — Установить инструменты (5 мин)

### Установить Node.js
1. Зайти на https://nodejs.org
2. Скачать версию **LTS** (рекомендуемую)
3. Установить — нажимать «Далее» везде
4. Проверить: открыть **Командная строка** (Windows) или **Терминал** (Mac)
   и ввести:
   ```
   node --version
   ```
   Должно показать: `v20.x.x` или похожее ✅

### Установить VS Code (редактор кода)
- Скачать на https://code.visualstudio.com — бесплатно

---

## ШАГ 2 — Создать проект (3 мин)

Открыть терминал и выполнить по очереди:

```bash
# Создать папку проекта
mkdir msb-credit && cd msb-credit

# Создать Vite + React проект
npm create vite@latest . -- --template react

# Установить зависимости
npm install

# Установить дополнительные пакеты
npm install xlsx vite-plugin-pwa workbox-window
```

---

## ШАГ 3 — Скопировать файлы приложения (5 мин)

Скопировать файлы из этого архива в папку проекта:

```
msb-credit/
├── index.html          ← заменить
├── vite.config.js      ← заменить
├── package.json        ← заменить
└── src/
    ├── main.jsx        ← заменить
    └── App.jsx         ← заменить (основной код приложения)
```

---

## ШАГ 4 — Добавить иконки приложения (3 мин)

1. Создать папку `public/icons/` в проекте
2. Добавить два файла иконок:
   - `icon-192.png` — 192×192 пикселей (логотип банка/отдела)
   - `icon-512.png` — 512×512 пикселей (тот же логотип, больше)

   > Можно использовать любое изображение. Сделать иконку онлайн: https://www.pwabuilder.com/imageGenerator

3. Создать файл `public/favicon.svg`:
   ```svg
   <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
     <rect width="100" height="100" rx="20" fill="#E8A020"/>
     <text y=".9em" font-size="80" x="10">🏦</text>
   </svg>
   ```

```bash
npm run dev
```

Всё работает? Отлично! ✅

## ШАГ 6 — Опубликовать на Vercel (10 мин)

### 6.1 — Создать аккаунт на GitHub
- Зайти на https://github.com → Sign Up (бесплатно)

### 6.2 — Загрузить проект на GitHub
```bash
# В папке проекта:
git init
git add .
git commit -m "МСБ Кредит PWA"
git branch -M main

# Создать репозиторий на GitHub (github.com → New repository)
# Назвать: msb-credit
# Затем выполнить (заменить USERNAME на ваш логин):
git remote add origin https://github.com/USERNAME/msb-credit.git
git push -u origin main
```

### 6.3 — Подключить Vercel
1. Зайти на https://vercel.com → Sign up with GitHub
2. Нажать **Add New Project**
3. Выбрать репозиторий `msb-credit`
4. Нажать **Deploy**

Через 2 минуты приложение будет доступно по адресу:
`https://msb-credit.vercel.app` ✅

---

## ШАГ 7 — Установить на телефон

### iPhone (iOS):
1. Открыть Safari (именно Safari, не Chrome!)
2. Перейти на адрес приложения
3. Нажать кнопку «Поделиться» (квадрат со стрелкой вверх)
4. Выбрать **«Добавить на экран "Домой"»**
5. Нажать **«Добавить»**

### Android:
1. Открыть Chrome
2. Перейти на адрес приложения
3. Нажать **⋮ меню** → **«Установить приложение»** или **«Добавить на главный экран»**
4. Нажать **«Установить»**
